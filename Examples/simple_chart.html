This example assumes that two numeric values beetween 0-255 come from serial device periodically. Values need to be separated by empty space and there must be EOL (line change) at the end. These two values are used to draw simple chart. Chart is only updated when new data arrives.

Example also shows how to parse imaginary 'Command' lines from incoming data. It expects to receive lines like :

Command X received.

This data is not used, but is displayed on dataLog.

Example project for Arduino is included, it manipulates values based on keycode received.

Host is localhost:5331 (change this from .js if you need to).
